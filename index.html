Icephys Dandiset Visualization

This is a visualization of some of the intracellular electrophysiology (icephys) data found across the open neuroscience initiative DANDI. The data is visualized using a UMAP and a parallel coordinates plot. The data is also visualized in a table format. this is currently a work in progress and is not yet complete. Please cite the original authors of the data when using this data.
